# Lógica de programação

## Conceitos

- computadores são extremamente literais, ou seja, tenho que informar todos os passos para que ele consiga executar uma determinada tarefa.
- algoritmos: conjunto de instruções (passos) finitos e organizados que, ao serem executados, resolvem um problema
    - estrutura: entrada -> processamento -> saída
    - exemplo: calcular a média das notas de alunos de uma turma que realizaram 4 provas (p1, p2, p3, p4)
        - entrada: nota p1, nota p2, nota p3, nota p4 (para cada aluno)
        - processamento: 
            - olhar o valor de p1;
            - inserir o valor de p1;
            - olhar o valor de p2;
            - inserir o valor de p2;
            - olhar o valor de p3;
            - inserir o valor de p3;
            - olhar o valor de p4;
            - inserir o valor de p4;
            - somar os valores inseridos (p1 + p2 + p3 + p4)
            - dividir o resultado da soma por 4
        - saída: média
